Program Logic


1.Get the JSON from the web service given.
2.Parse the JSON and create objects for each owner(Each owner object can have list of pets).
3.Create a list to include all the owner objects.
4.Group the List by gender.
5.Combine the list of pets of same gender.(This is to do the sort operation across all the list of pets in same gender)
6.Filter the cats from the list and sort it in alphabetical order
7.Display the list of cats grouped by gender of the owner.
